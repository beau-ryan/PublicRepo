## 3. Getting Started

### Prerequisites

- Windows PowerShell 5.1 or later
- Python 3.11.9 or later (for full functionality)
- Git (repository is version controlled)

### Quick Start

1. Open PowerShell as Administrator (recommended)
2. Navigate to the project directory: `cd i:\USOS`
3. Run the main script: `.\Platforms\USOS.ps1`
4. Follow the interactive menu to set up your environment

### Key Script Features

The USOS.ps1 script is a comprehensive automation tool that:

- **Version**: 0.0.2

- **Full Name**: Universal Sovereign Operation Software Orchestration
- **Type**: PowerShell-based automation suite
- **Menu Options**: 24 different operations for complete system management

### Referenced Project Paths

The script references several project components that appear to be part of a larger system:

- `I:\USOS\Universal_RunnerApp\` - Main application directory
- `I:\USOS\.SophiaPY` - Python virtual environment
- `I:\USOS\Env_library\` - Environment configuration files
- `I:\USOS\SYS_Root_Admin\` - Administrative scripts

*Note: These directories are referenced in the script but may not be present in the current repository state.*
